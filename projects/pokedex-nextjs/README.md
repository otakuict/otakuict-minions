# Pokedex Next.js

Educational Pokedex-style web app built with Next.js and the public PokeAPI.

## Run

```powershell
npm.cmd install
npm.cmd run dev
```

Open:

```text
http://localhost:3000
```

## Features

- Open and close Pokedex shell interaction.
- Search Pokemon by name or ID.
- Displays sprite, types, stats, height, weight, and abilities.
- Uses the public Pokemon API from `https://pokeapi.co/api/v2/pokemon/{name-or-id}`.
